Introduction

Computer Graphics (CG) is a field of computer science that focuses on the creation, manipulation, and representation of visual images using computational techniques.

Types of Computer Graphics

Computer graphics can be classified into two main types: Raster Graphics and Vector Graphics.

Applications of Computer Graphics

Computer graphics are widely used in various domains such as gaming, simulations, medical imaging, and more.

Basic Techniques in CG

Some basic techniques include rendering, ray tracing, and rasterization.
